solo game + mode #3

Suppose players P1 and P2 have been matched together.

Players finishe match normally

First consider the scenario where the match finishes normally.

Players fail to finish the match

Now consider the following scenarios where players fail to finish the match.

Scenario 1: P2 finishes the match normally but P1 left the room without finishing the game (e.g., click “abort” button or force quit)

Scenario 2: P2 finishes the match normally; P1 stays in the room but fails to submit score before maxMatchDuration

Scenario 3: both P1 and P2 fail to finish the match (i.e., both players leave the room without finishing the game, or both players fail to finish the game before maxMatchDuration). In this case, the match is a draw and entry fee will be returned to both players.

Exchange state (optional)

In some use cases, players may need to exchange auxiliary information even in solo game competitions (e.g., a player may want to see his opponent’s real-time game score).

Note that state exchange is optional for solo game and signature is not required.
